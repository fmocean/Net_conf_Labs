# Net_conf_Labs

Net_conf_Labs is my collection of practical network configuration labs. It’s a place to park real configs, topologies, and notes from home lab and study work so I can revisit and expand them over time.

Right now there are three main lab areas, each in its own branch:

- `HomeLabs` – home lab topologies and configs  
  https://github.com/fmocean/Net_conf_Labs/tree/HomeLabs

- `iperf3_T` – iperf3 traffic and throughput testing labs  
  https://github.com/fmocean/Net_conf_Labs/tree/iperf3_T

- `BGP_LABS` – BGP-focused routing and policy labs  
  https://github.com/fmocean/Net_conf_Labs/tree/BGP_LABS

More lab branches and topics will be added as I build and test new scenarios.

To use this repo, clone it, switch to the branch you’re interested in, and follow the README and files in that branch:

```bash
git clone https://github.com/fmocean/Net_conf_Labs.git
cd Net_conf_Labs
git checkout HomeLabs    # or iperf3_T, or BGP_LABS
```

Each branch is self-contained, with its own structure, configs, and notes.
